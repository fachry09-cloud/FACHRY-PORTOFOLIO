# FACHRY-PORTOFOLIO
Teknologi Informasi (IT) adalah bidang yang mencakup penggunaan komputer, perangkat lunak, jaringan, dan sistem digital untuk mengelola, menyimpan, mentransmisikan, dan mengamankan data. IT berperan penting dalam mendukung operasional, komunikasi, dan pengambilan keputusan di berbagai sektor kehidupan.
